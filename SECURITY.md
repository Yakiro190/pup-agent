# Security Policy

## Supported Versions

This project currently supports security updates for:

- `0.1.x` (latest `main`)

## Reporting a Vulnerability

Please **do not** open public issues for security vulnerabilities.

Use GitHub: **Security → Report a vulnerability** (private advisory):

- https://github.com/Yakiro190/pup-agent/security/advisories/new

Please include:
- vulnerability description,
- impact,
- reproducible steps or PoC,
- suggested fix (if available).

We aim to acknowledge reports within **72 hours** and provide status updates
until resolution.
